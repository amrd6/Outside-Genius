Welcome to Outside-Genius.

Outside-Genius is an android application which allows fans at festivals like Outside Lands (5th Aug - 7th Aug) to listen to all the artists performing at the event for 10-15 seconds and choose which one they like the best to make a clash-free schedule for the 3 day festival.

Key features: 1. Allows to listen to most popular track of each artist. 2. Allows to create a schedule for the event. 3. Notifies if there are any clashes in the two chosen artist's performance. 4. Let fans navigate from one event to the other. 5. Easy dashboard style UI. 6. Few clicks and you have your schedule ready.
